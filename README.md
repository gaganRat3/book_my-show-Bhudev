# Book My Show Clone (Django)

This project is a ticket booking website inspired by Book My Show, built with Django (Python) for the backend and HTML, CSS, and JavaScript for the frontend.

## Features

- User authentication (Sign up, Login)
- Event listing page
- Event selection and seat arrangement
- Seat booking and confirmation

## Structure

- `bookmyshow/` - Django project core
- `static/` - Frontend assets (HTML, CSS, JS)
- `templates/` - Django HTML templates

## Setup

1. Ensure Python 3.14+ is installed.
2. Install Django: `pip install django`
3. Run migrations: `python manage.py migrate`
4. Start server: `python manage.py runserver`
